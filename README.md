# Personal Portfolio Website

A responsive personal portfolio website built using HTML, CSS, and JavaScript.

## Preview

This website showcases my profile, technical skills, projects, education, and contact information in a clean and modern interface.

## Features

- Responsive Design
- Dark / Light Mode
- Fixed Navigation Bar
- Smooth Scrolling
- Back to Top Button
- Modern Card Layout
- Mobile Friendly
- Simple and Clean UI

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)
- Font Awesome
- Google Fonts (Poppins)
